# 2D Computer graphics met Java2D

Dit is een lessenpakket over Java2D. Deze documentatie is beschikbaar op [github.io](https://avansti.github.io/TI1.3-2DGraphics-Lesmateriaal/). In een klein aantal lessen worden basistechnieken van Java2D beschreven, waarna een aantal toepassingen volgen. Bij dit lessenpakket horen een aantal IntelliJ projecten met voorbeeld-broncode die je kunt gebruiken om op verder te bouwen. Deze kun je vinden op [github](https://github.com/AvansTi/TI1.3-2DGraphics-Voorbeelden)

## lessen

- [Les 1 - Java2D, Lijnen, Kleuren, Basistransformaties](week1)
- [Les 2 - Shapes, Paths, Areas, Strokes, Paints, Transformaties en interacties](week2)
- [Les 3 - Tekst, Images, Blending, Clipping, Animeren](week3)
- [Les 4 - Verlet particles](week4)
- [Les 5 - Box2D Physics Engine](week5)
- [Les 6 - Flappy Bird](week6)
- [Les 7 - Platformer game](week7)
